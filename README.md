AI/ML Portfolio — Vishwas Vijay Bhattgaonkar

Files:
- index.html — main portfolio (two-panel premium design)
- style.css — styles and animations
- script.js — tab handling, animations, accessibility
- resume.html — print-friendly resume
- assets/ — profile svg and future assets

Quick preview
1. Open `index.html` in a browser to preview the portfolio locally.
2. Click tabs (About / Skills / Featured Projects). Use arrow keys to navigate tabs.
3. Open `resume.html` and use Print → Save as PDF for a resume PDF.

Deploy
- GitHub Pages: push this folder to a repository and enable Pages from `main` branch root.
- Netlify: drag & drop the folder to Netlify or connect the repo.

Notes
- All code is framework-free, semantic, and optimized for fast loading.
- Assets can be replaced with real profile images or PDFs in `assets/`.
